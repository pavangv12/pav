## 🏗️ Installation & Setup  

Want to run this locally? Follow these steps:  

1. Clone the repository:  
   ```bash
   git clone https://github.com/pavangv12/pav.git  
   cd pav  
   ```  
2. Install dependencies:  
   ```bash
   npm install  
   ```  
3. Start the development server:  
   ```bash
   npm start  
   ```
